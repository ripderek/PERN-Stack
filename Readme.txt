La carpeta client es el frontend que esta iniciado con vite
Las carpetas que estan fuera del cliente son el backend 
.env contine los parametros para conectar a la base de datos 
.gitignore es para no se suban ciertos archivos al github
.vscode contiene un json que se llama settings alli estan en objetos los archivos que se quieren ocultar (no borrar) del vs code como por ejemplo la carpeta modules, esto se hace para que vs code no carge los archivos en el editor y no se ponga lento xd
